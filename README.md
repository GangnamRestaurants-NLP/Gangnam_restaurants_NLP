## 🍚 Intro
- 신입의 경우 회사 근처 식당을 잘 모르기 때문에 회식장소를 선정해야 할 때 고민이 많을 것이라고 생각됩니다.
그래서 저희는 회식장소를 선정할 때 고려해야 할 사항을 고민해봤습니다.</br></br>
